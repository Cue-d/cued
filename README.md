# PRM - Personal Relationship Manager

A cloud-based personal CRM with multi-platform messaging (iMessage, Gmail, Slack) and AI-powered action suggestions.

## Prerequisites

| Requirement | Version | Check |
|-------------|---------|-------|
| Node.js | 22+ | `node --version` |
| pnpm | 9+ | `pnpm --version` |

**For Electron app (macOS only):**
- macOS 15+, Swift 6.0+ (`swift --version`)
- Full Disk Access and Contacts permissions in System Settings

## Quick Start

```bash
# 1. Install dependencies
pnpm install

# 2. Set up Convex backend
cd packages/convex
npx convex login          # One-time browser auth
npx convex dev            # Creates .env.local with deployment URL
# Note the deployment URL (e.g., dev:your-project-123)

# 3. Set up web app
cd ../../apps/web
cp .env.local.example .env.local
# Edit .env.local - see Environment Variables below

# 4. Start everything
cd ../..
pnpm dev
```

Web app opens at http://localhost:3000

## Environment Variables

Each package has its own `.env.local` file (gitignored).

### packages/convex/.env.local

Auto-generated by `npx convex dev`:

```bash
CONVEX_DEPLOYMENT=dev:your-project-name
```

### apps/web/.env.local

```bash
# Required: Convex (convert deployment to URL)
# dev:your-project-name → https://your-project-name.convex.cloud
NEXT_PUBLIC_CONVEX_URL=https://your-project-name.convex.cloud

# Required: WorkOS Auth
WORKOS_CLIENT_ID=client_xxx
WORKOS_API_KEY=sk_xxx
WORKOS_COOKIE_PASSWORD=32-char-random-string
NEXT_PUBLIC_WORKOS_REDIRECT_URI=http://localhost:3000/callback

# Optional: Integrations
NANGO_SECRET_KEY=xxx          # Gmail/Slack sync (see NANGO_SETUP.md)
OPENAI_API_KEY=sk-xxx         # AI features
MEM0_API_KEY=xxx              # Memory features
```

### apps/electron/.env.local (if needed)

```bash
# Uses same Convex URL as web
CONVEX_URL=https://your-project-name.convex.cloud
```

## Scripts

| Script | Description |
|--------|-------------|
| `pnpm dev` | Start all apps in dev mode |
| `pnpm build` | Build all packages |
| `pnpm lint` | Lint all packages |
| `pnpm typecheck` | Type check all packages |
| `pnpm test` | Run all tests |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "Error accessing messages database" | Grant Full Disk Access to your terminal |
| "Contacts access denied" | Grant Contacts access in System Settings |
| Convex types missing | Run `cd packages/convex && pnpm dev` |
